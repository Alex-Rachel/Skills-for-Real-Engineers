# 开发中

仍在开发中的技能。它们尚未准备好发布——可能会有粗糙的边缘、破坏性变更和被放弃的实验。在它们晋升到稳定的分类之前，不会出现在插件和顶层 README 中。

- **[review](./review/SKILL.md)** — 沿两条平行轴审查自某个固定点以来的变更：**标准**（差异是否符合仓库的编码规范？）和 **规格**（差异是否忠实地实现了原始 issue/PRD？）。
- **[writing-beats](./writing-beats/SKILL.md)** — 以"节拍"之旅的形式塑造文章，类似选择你自己的冒险风格。选择一个起始节拍，只写该节拍，然后转向下一个，直到文章自然收尾。
- **[writing-fragments](./writing-fragments/SKILL.md)** — 通过追问式对话挖掘你的碎片——异质的写作素材——并将它们追加到同一篇文档中，作为未来文章的原始素材。
- **[writing-shape](./writing-shape/SKILL.md)** — 取一份包含原始素材的 markdown 文件，逐段将其塑造为文章，并在每一步论证格式选择的理由。
- **[teach](./teach/SKILL.md)** — 在当前工作区教授用户新技能或概念。通过使命驱动、知识-技能-智慧三阶段教学，辅以 HTML 讲解器、互动练习和社区推荐。
