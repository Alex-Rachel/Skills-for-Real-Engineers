# GLOSSARY.md 格式

`GLOSSARY.md` 是本教学工作区的规范语言。所有讲解器、练习和学习记录应遵循其术语。构建词汇表本身就是学习的一部分：将一个概念压缩为紧凑定义是用户理解它的证据。

## 结构

```md
# {主题} 词汇表

{一两句话描述本词汇表覆盖的主题。}

## 术语

**Hypertrophy（肌肥大）**:
由机械张力和代谢压力在重复训练过程中驱动的肌肉增长。
_Avoid_: Bulking, getting big

**Progressive overload（渐进超负荷）**:
通过负荷、容量或强度，系统性地随时间增加对肌肉的需求。
_Avoid_: Pushing harder, levelling up

**RPE（自觉用力程度评级）**:
1-10 的自我评分，表示一组动作感觉有多难，10 是力竭，8 意味着还有两个 reps 的余量。
_Avoid_: Effort score, intensity rating
```

## 规则

- **只在用户理解术语后才添加。** 词汇表是压缩知识的记录，不是用户用来学习的字典。如果用户刚刚接触一个概念，等到他们能正确使用它之后再提升到这里。
- **要有主见。** 当同一概念存在多个词时，选最好的那个，将其余的列为要避免的别名。这是语言压缩的方式。
- **定义要紧凑。** 一两句话。定义术语"是什么"，而不是它"做什么"或"怎么做"。
- **在定义内部使用词汇表自己的术语。** 一旦术语进入词汇表，就在所有地方优先使用它——包括其他定义内部。这使得复杂术语后来更容易理解。
- **在自然分群出现时用子标题分组。** 平铺列表在术语内聚时即可。
- **明确标注歧义。** 如果某个术语在更广泛的领域中被松散使用，注明解决方案："在本工作区中，'set' 始终指工作集——热身单独追踪。"
- **随着理解加深而修订。** 用户在第一周写的定义可能在第六周是错的。原地更新；不要留下过时条目。
