# 分流标签

技能使用五个标准分流角色。此文件将这些角色映射到你的 issue 跟踪器中实际使用的标签字符串。

| mattpocock/skills 中的标签 | 我们的跟踪器中的标签 | 含义                                   |
| -------------------------- | -------------------- | --------------------------------------- |
| `needs-triage`             | `needs-triage`       | 维护者需要评估此 issue                 |
| `needs-info`               | `needs-info`         | 等待报告者提供更多信息                 |
| `ready-for-agent`          | `ready-for-agent`    | 已完整规格化，可供 AFK agent 处理      |
| `ready-for-human`          | `ready-for-human`    | 需要人工实现                           |
| `wontfix`                  | `wontfix`            | 不会处理                               |

当技能提到一个角色时（例如"应用 AFK-ready 分流标签"），使用此表中对应的标签字符串。

编辑右列以匹配你实际使用的词汇。
